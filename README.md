# Student Academic Performance Prediction using Machine Learning

## Overview
Developed a machine learning model to predict students’ academic performance using demographic and educational factors such as gender, parental education, lunch type, and test preparation status.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Machine Learning
- Random Forest Regression

## Workflow
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature encoding and scaling
- Model training and evaluation

## Result
Built a Random Forest Regression model and evaluated performance using MAE, RMSE, and R² score to generate accurate score predictions.
